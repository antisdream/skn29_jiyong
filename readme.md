# SKN 29기 학습 기록

SKN 29기 수업 과정에서 작성한 실습 노트와 예제 코드를 날짜별로 정리한 저장소입니다. 각 폴더는 수업 일차를 기준으로 구성되어 있으며, 학습 흐름을 다시 확인하거나 주요 실습 내용을 복습하기 위한 목적으로 관리합니다.

## 저장소 구성

- `day01` ~ `day12`: 프로그래밍과 데이터 기초

수업 자료 중 개인 설정, 인증 정보, 원본 데이터, 대용량 모델 파일, 캐시 파일, 임시 산출물은 저장소에 포함하지 않습니다.
