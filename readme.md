## 구성 원칙

수업 진행 순서에 맞춰 일차별 실습 자료를 정리합니다. 폴더와 파일은 커리큘럼 진행 상황에 따라 추가되거나 조정될 수 있으며, README는 전체 학습 흐름을 이해하기 위한 요약 문서로 관리합니다.

저장소에는 학습 내용을 확인하는 데 필요한 코드, 노트북, 정리 문서만 선별해 포함합니다. 실행 과정에서 생성되는 산출물이나 개인 환경에 의존하는 파일은 공개 대상에서 제외합니다.

## 공개 기준

다음 항목은 저장소에 포함하지 않습니다.

- 인증 정보, API 키, 토큰, 비밀번호
- 개인 환경 설정과 로컬 실행 전용 파일
- 원본 데이터셋, 대용량 산출물, 모델 체크포인트
- 캐시, 로그, 임시 파일, 개인 참고 문서

## 활용 안내

일부 실습은 외부 데이터, 별도 환경 변수, 로컬 실행 환경을 전제로 작성되었습니다. 따라서 모든 파일이 단독으로 바로 실행되는 것을 목표로 하기보다는, 수업 과정에서 다룬 개념과 코드 흐름을 확인하는 용도로 구성되어 있습니다.

## 사용 범위

본 저장소는 학습 기록과 포트폴리오 참고 목적으로 공개합니다. 수업 자료, 외부 데이터, 모델 파일, 제3자 저작물은 포함하지 않으며, 사용된 라이브러리와 외부 리소스의 라이선스는 각 원 출처의 정책을 따릅니다.
